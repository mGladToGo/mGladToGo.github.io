React-Node Web App with PostgreSQL database

Having completed an online course about web development, my next goal was to create a simple web app that utilised everything that I had learned, this meant taking a component of the course used to generate list items entered by the user and give it full-stack functionality, utilising logins, database storage, and the interactive components of React. NOTE: The app is currently stored on render and takes time to spin up after not being used, on opening backend code will take ~30 seconds to be able to be run properly.
